# Global CO₂ Emissions Analytics Dashboard (2018 vs 2020)

A Python data analytics platform designed to visualize and evaluate global per-capita CO₂ emissions across two key baseline timelines: 2018 (pre-pandemic) and 2020 (peak COVID-19 lockdown cycles).

## 📊 Dashboard Preview
![Dashboard Preview](carbon_dashboard.jpg)

## 🛠️ Tech Stack & Tools Used
- **Python** (Core Logic)
- **Pandas** (Data Aggregation, Grouping, and Manipulation)
- **Plotly & Plotly Express** (Interactive Canvas Visualizations)
- **Plotly Subplots** (Grid Structural Layout Mapping)

## 📌 Key Findings
- **Highest Emitter:** Qatar tracks the peak emission intensity at 31.73 t per capita.
- **Continental Share:** Asia dominates global emissions (~38.9%), closely followed by Europe (~31.6%).
- **Economic Disparity:** High-Income nations carry the largest cumulative share of emissions (~421.2 Mt).
- **Pandemic Impact:** The bottom scatter plot isolates a widespread drop in emission percentages during 2020 lockdown windows.
